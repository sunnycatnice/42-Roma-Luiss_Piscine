Echo
Echo stampa a display il testo che viene scritto successivamente
Echo può anche stampare il valore contenuto della variabile 
ESEMPIO: 	echo nome —> stampa nome
		echo $nome —> stampa il valore della variabile nome
Echo ~ (option 5) serve per sapere la cartella root senza conoscere a priori il nome.

LS
ls LISTA cioè stampa il contenuto della cartella attuale.
Ls -l LISTA (long) cioè stampa i permessi di ogni file della cartella attuale (vedi chmod per cambiare permessi)
Ls -ld LISTA (directory) lista le informazioni di una directory
Ls | (pipe) reindirizza le istruzioni ad un altro programma.
Ln -s cartella0 cartella6 —> crea un link della cartella0 alla cartella6
ESEMPIO: 	ls | grep txt —> filtra tutti i file e trova solo quelli di testo.
		ls -lt —> ordina nel tempo i file listati.
